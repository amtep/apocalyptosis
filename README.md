# Apocalyptosis

A mix of Apocalypse, Apotheosis, and a disease.

This will be a game about running a secret cult in the modern world.
Recruit priests, mooks, and goons.
Develop bases around the world.
Discover deeper secrets of the deeper truths.
Stay hidden.

There's nothing playable here yet, though.
We're just setting up the basic framework of the game.

# Inspiration

The game design takes a lot of inspiration from [Endgame: Singularity](https://github.com/singularity/singularity).
Do check out that game!
